# 🏠 Property Rental & Booking Platform

A modern full-stack Property Rental & Booking Platform where property owners can list rental properties and tenants can discover, book, pay reservation fees, save favorites, and leave reviews. The platform includes secure authentication, role-based access control, Stripe payment integration, and dedicated dashboards for Tenants, Owners, and Admins.

##  Project Purpose

The purpose of this project is to create a secure and user-friendly rental marketplace that connects property owners with tenants.

The platform allows:

* Property owners to manage rental listings.
* Tenants to search, book, and pay online.
* Admins to moderate users, properties, and bookings.
* Secure authentication using JWT.
* Online payment through Stripe.
